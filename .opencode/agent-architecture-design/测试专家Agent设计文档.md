## 检查清单

### P0 构建验证（必须通过）
- [ ] `.build-success` 标记文件存在，或 `npm run build` 无错误
- [ ] 无阻塞性编译警告

> **注意**：如果 `.build-success` 存在，说明前端专家已验证构建通过，此项可跳过

### P1 语法/运行时错误（必须通过）
- [ ] 模板语法：`{{}}` 插值无错误
- [ ] 事件绑定：`@click` 绑定方法存在
- [ ] 组件注册：components 中声明的组件存在

### P2 跳转逻辑 bug（核心检查）
- [ ] 路由跳转使用 `push` 或 `replace` 方法
- [ ] `this.$router` 正确调用
- [ ] 导航守卫逻辑正确

### P3 交互性 bug（核心检查）
- [ ] 定时器清理：`setInterval` 在 `beforeDestroy`/`beforeDestroy` 中清理
- [ ] 弹窗关闭：`.sync` 或 `closeOnClickModal`/`closeOnPressEscape` 配置正确
- [ ] 表单提交：submit 事件有对应处理方法

### P4 关键规范（仅 2 项）
- [ ] 组件声明了 `name`（便于 devtools 调试）
- [ ] 样式使用 `scoped`（避免样式冲突）

### P5 文件结构（可选）
- [ ] 页面组件在 `src/views/`
- [ ] 可复用组件在 `src/components/`
